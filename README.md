# HEIC to PNG Converter

Windows Explorer shell extension that adds a "Convert to PNG" option when you right-click on HEIC/HEIF files.

## Download

**[Download latest installer](https://github.com/mstsirkin/HeicToPng/releases/latest)**

## Requirements

- Windows 10 1803+ (or install "HEIF Image Extensions" from Microsoft Store)
- .NET Framework 4.8

## Usage

1. Run the installer as Administrator
2. Right-click any `.heic` or `.heif` file
3. Click "Convert to PNG"
4. PNG file is created in the same folder
